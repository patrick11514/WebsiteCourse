Display

`block` zabere celou šířku, `inline` respektuje šířku obsahu, `inline-block` je kombinace (lze mu dát šířku/výšku).

block
inline A inline B
inline-block

Position

`absolute` se bere vůči nejbližšímu rodiči s `position: relative` (jinak k oknu). `fixed` je k oknu vždy, `sticky` přepíná mezi statickým a „přilepeným“.

static
relative
absolute

Scrolluj kousek dolů a sleduj „fixed“ vpravo dole.
